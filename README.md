---

## 📥 Descargas (Windows)

Elige la opción que mejor se adapte a ti. **Recomendamos usar el "Instalador Automático"**.

| Tipo de Descarga | Descripción | Enlace |
| :--- | :--- | :---: |
| 📦 **Versión Portable** | No requiere instalación. Ideal para llevar en un USB. | [**Descargar Portable**](https://github.com/acierto-incomodo/repo-game/releases/latest/download/Launcher_Portable.exe) |
| 💿 **Instalador Clásico** | El instalador estándar del launcher. | [**Descargar Installer**](https://github.com/acierto-incomodo/repo-game/releases/latest/download/REPO_Launcher_Installer.exe) |

> **Nota:** El launcher requiere una conexión a Internet activa para descargar los archivos del juego la primera vez.
